Description:

This webpage was created using Django as a part of one of the assignments. The website is all about generating random passwords for the user.

Functionalities:

If nothing is specified, a password of length 12 having all small characters would be generated.

Or You can utilize the following functions:

1. Choose the length of the password from the dropdown
2. Check the 'UpperCase' checkbox if you want to include UpperCase characters
3. Check the 'Numbers' checkbox if you want to include numbers
4. Check the 'Special Characters' checkbox if you want to include SpecialCharacters
